  • asp.net中使用fckeditor

    看看topcal的这篇文章，基本动作就会了
    步骤大致如下：
    1. 引用fckeditor的dll文件，或者下载fckeditor.net，将文件添加到项目中，然后引用该项目，建议下载源码添加到项目中，这样可以修改，且不需要在工具箱中再添加了。
    2. 在web.config中的appsettings段加入如下声明
    <appSettings>
    <add key="FCKeditor:BasePath" value="~/FCKeditor/"/>
    <add key="FCKeditor:UserFilesPath" value="/Files/" />
    </appSettings>
    3. 将fckeditor的文件copy到web项目的目录FCKeditor目录中，修改目录中的FCKconfig.js，将该文件中的指定部分修改如下：
    var _FileBrowserLanguage = 'aspx' ;
    var _QuickUploadLanguage = 'aspx' ;
    _FileBrowserLanguage = 'aspx' ;
    _QuickUploadLanguage = 'aspx' ;
    4. 新建一个文件，将工具栏中的fckeditor拖放到页面中运行
