修改浏览器的user-agent字符串,模拟移动设备访问站点

修改浏览器的user-agent字符串, 模拟mobile设备访问资源,譬如模拟iphone/ipad访问现在的126油箱, 和新出的狗狗怕辣死.  chrome之前有插件可以实现修改, 但新版本中好像已经不支持.推荐使用firefox或safari浏览器.

1. ie需要修改注册表

IE的user-agent取值下面是下列地址的组合+一些默认值

HKCU\Software\Microsoft\Windows\CurrentVersion\Internet Settings\User Agent\Post Platform 
HKLM\Software\Microsoft\Windows\CurrentVersion\Internet Settings\User Agent\Post Platform 
HKLM\Software\Microsoft\Windows\CurrentVersion\Internet Settings\5.0\User Agent\Post Platform

 修改后重启ie即可

2. opera 这个没有深究, 看搜索结果上好像不能随意修改, 只有部分固定值可选; 

3. safari打开develop菜单, 可以直接修改user-agent, 模拟其他浏览器, 也内置很多常用user-agent,

4. firefox上有插件user agent switcher可以比较方便修改user agent字符串, 配合firebug等插件, 效率较高.

下面是一些user agent字符串, firefox插件中需要添加(也可以直接从safari中复制)

iphone3.0 "Mozilla/5.0 (iPhone; U; CPU iPhone OS 3_0 like Mac OS X; en-us) AppleWebKit/528.18 (KHTML, like Gecko) Version/4.0 Mobile/7A341 Safari/528.16" 

iPhone 4.0 "Mozilla/5.0 (iPhone; U; CPU iPhone OS 4_0 like Mac OS X; en-us) AppleWebKit/532.9 (KHTML, like Gecko) Version/4.0.5 Mobile/8A293 Safari/6531.22.7"

iPad2 "Mozilla/5.0(iPad; U; CPU OS 4_3 like Mac OS X; en-us) AppleWebKit/533.17.9 (KHTML, like Gecko) Version/5.0.2 Mobile/8F191 Safari/6533.18.5"

iPod Touch "Mozilla/5.0 (iPod; U; CPU iPhone OS 4_1 like Mac OS X; en-us) AppleWebKit/532.9 (KHTML, like Gecko) Version/4.0.5 Mobile/8B117 Safari/6531.22.7"

 Android的user agent字符串, 请参考Google Android user agent strings

Google Nexus One

Mozilla/5.0 (Linux; U; Android 2.2; en-us; Nexus One Build/FRF91) AppleWebKit/533.1 (KHTML, like Gecko) Version/4.0 Mobile Safari/533.1

Mozilla/5.0 (Linux; U; Android 2.1; en-us; Nexus One Build/ERD62) AppleWebKit/530.17 (KHTML, like Gecko) Version/4.0 Mobile Safari/530.17

HTC

Mozilla/5.0 (Linux; U; Android 2.1-update1; de-de; HTC Desire 1.19.161.5 Build/ERE27) AppleWebKit/530.17 (KHTML, like Gecko) Version/4.0 Mobile Safari/530.17
