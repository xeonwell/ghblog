安装Nginx为Windows服务

用到微软的2个工具: instsrv.exe srvany.exe
第一步:运行命令

instsrv Nginx d:\nginx\srvany.exe

这样就安装了一个Nginx的服务

第二步:配置Nginx的运行参数
直接将配置导入到注册表

Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NGINX\Parameters]
"Application"="d:\\nginx\\nginx.exe"
"AppParameters"=""
"AppDirectory"="d:\\nginx\\"

在实际运用中，利用M$的工具安装为服务随机自动启动还是不错的，不过reload和quit就不太理想，建议还是命令行操作。
两个工具奉上，也可以到官网下载 http://www.microsoft.com/downloads/details.aspx?FamilyID=9D467A69-57FF-4AE7-96EE-B18C4790CFFD&displaylang=en
