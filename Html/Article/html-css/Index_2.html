  • 设置图片上的工具栏不显示

    使用ie访问一些网站,经常会在图片上出现一个工具栏,极其破坏感觉,以前我是用js脚本给图片添加galleryImg属性:img.galleryImg = false | no;今天闲逛,看到了这个方法
    <meta http-equiv="imagetoolbar" content="no" />
    使用该标签可以禁止所有图片上的工具栏.方便且效率多了.
    当然,个别图片上允许出现,我们还是可以通过设置单个图片的galleryimg属性来控制.

  • IE条件注释参考

    基本例子：

    <!--[if IE]>仅IE可见<![endif]-->
    <!--[if IE 5.0]>仅IE 5.0可见<![endif]-->

    参数属性：

    gt : greater than，选择条件版本以上版本，不包含条件版本
    lt : less than，选择条件版本以下版本，不包含条件版本
    gte : greater than or equal，选择条件版本以上版本，包含条件版本
    lte : less than or equal，选择条件版本以下版本，包含条件版本
    ! : 选择条件版本以外所有版本，无论高低
    <!--[if gt IE 5.0]>仅IE 5.0以上可见<![endif]-->
    <!--[if lt IE 5.0]>仅IE 5.0以下可见<![endif]-->
    <!--[if gte IE 5.0]>IE 5.0及以上可见<![endif]-->
    <!--[if lte IE 5.0]>IE 5.0及以下可见<![endif]-->
    <!--[if lte IE 5.0]>非IE 5.0可见<![endif]-->
    <!--[if lt IE 5]>仅IE 5.0和IE5.5可见<![endif]-->

  • 让页面上的小字体(10px,11px)清晰显示

    我们在页面中使用小字体时(11px和10px), 如果使用宋体, 会很不清晰, 所幸的是, 前辈们已经指点了迷津, 譬如日文字体Mingliu和PMingliu可以清晰显示11px的中文, 所以我们定义如下两个class, 在需要的地方, 直接element的class属性中加入下面的classname.
    .font11 { font-size:11px; font-family:Mingliu,PMingliu;}
    .font10 { font-size:10px; font-family:MS UI Gothicu}
