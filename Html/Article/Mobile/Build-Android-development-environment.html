搭建Android2.1开发环境

1. 安装JDK5或更高版本（如果以前没有安过的话）。

2. 安装Eclipse3.4或更高版本。本文以Eclipse3.5.1为例。

3. 安装Android SDK。本文以2.1为例。 到http://androidappdocs.appspot.com/sdk/android-2.1.html这里下载 android-sdk_r05-windows.zip。 将压缩文件解压后，执行SDK Setup.exe启动Android SDK&AVD管理器。

    

     选择要使用的SDK平台，点击Install按钮开始下载所需的文件。网速慢的话，下载的时间比较长，不要勾选太多。选择的组件下载完成后会自动安装，安装完成后在Installed Packages中可以看到安装成功的组件。确保至少一个AndroidSDK  Tools和一个SDK Platform 被成功安装。API和例子是可选的组件，但也建议下载。Usb驱动包也建议下载。

    

     创建AVD。选择Virtual Devices, 点击New按钮创建AVD，输入AVD名称选择目标平台后点击Create AVD按钮即可。

      

    选择Virtual Devices,选择刚创建的AVD，点击Start…，检测模拟器是否正常工作。

    

4. 安装Android开发插件ADT0.9.6。启动Eclipse，选择Help > Install New Software，在对话框中点击 Add….按 钮，在Add Site 对话框中输入在线更新网站的名称，比如 “Android ADT  Plugin，在”Location”一栏中输入下面的地址：https://dl-ssl.google.com/android/eclipse/, 点击OK。 

     

    回到Available Software界面，现在可以看到增加了”Developer Tools”选项，选中这个选项旁的复选框，会自动选择这个选项的子选项Android DDMS和Android Development Tools。点击Next。最后同意许可证协议后点Finish。下载和安装完成后重启Eclipse。
