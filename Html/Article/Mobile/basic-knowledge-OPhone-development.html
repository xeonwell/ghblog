OPhone开发的基础知识

Q：OPhone是什么？

A：OPhone是中国移动大力推行的一种操作系统的统称，用来抗衡中国联通的iPhone，同时，也是中国移动3G（TD-SCDMA）的主打操作系统，以中国移动将近5亿的用户基础，这将会是一个庞大的市场。

Q：OPhone采用什么系统？

A：OPhone采用Google的Android操作系统，Android是一种基于Linux的手机操作系统。

Q：OPhone有哪些开发？

A：OPhone有两种开发方式，一种是应用程序的开发，实际上就是Java应用程序。还有另一种非常激动人心的开发，就是Widget，这种完全基于JCHX（Javascript + CSS + Html + Xml）的开发方式，通过API与OPhone进行交互，可以开发出简单漂亮的小工具应用

Q：开发OPhone，要使用哪些工具？

A：OPhone是基于Java进行开发，Eclipse是不可少的，当然我认为也可以使用其它的IDE，比如说Intellij IDEA或者My Eclipse之类的，虽然我没有试过，如果你觉得你是一个大拿的话，也可以尝试使用记事本或者UtralEdit之类的进行开发。然后就是要装一些SDK和插件了，关于这方面的详细，可以访问中国移动Ophone的开发网站：http://www.ophonesdn.com

Q：我需要掌握什么技能？

A：基本的Java知识还是要的，当然也不是必需的，比如我就不懂Java。但如果你不会Java的话，你应该有一些基本OO语言的开发经验，以我来看，开发OPhone不是一件太难的事。如果你需要开发Widget，对JCHX的熟悉是必不可少的，如果你精通，你将会开发出非常酷炫的应用来。

Q：如何做？

A：很简单，首先去ophonesdn.com去下载sdk，再下载eclipse3.4.2，然后照着上面一步一步做，把环境建好，开始你的OPhone之旅吧。

Q：OPhone和Android是一样吗？

A：我们可以这么理解，OPhone是Android的子集，或者说继承的Android，OPhone除了支持所有Android的API外，还支持自己独有的API

Q：开发OPhone和iPhone哪个更好？

A：Ah，这个问题，我比较难回答，不过我可以对比一下，iPhone在国外，OPhone在国内；iPhone要收99美元一年的开发费用，OPhone不收钱；iPhone使用Objective-c作为开发语言，而OPhone使用Java作为开发语言；iPhone要在Mac平台上开发，虽然也有Windows平台上的，但我不认为部署起来是一个轻松的活，Ophone可以在Mac、Linux、Windows上开发；OPhone可以开发基于JCHX的Widget，而iPhone不可以；iPhone在国外可以收到钱，OPhone在国内盗版比较严重
